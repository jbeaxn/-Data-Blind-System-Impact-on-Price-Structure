# 📊 도매시장 블라인드제 도입 전·후 거래 데이터 분석 🍎

Wholesale Market Blind System  
Data Collection / Analysis / Visualization Project

---

## 1. 프로젝트 개요 🎯

본 프로젝트는 **도매시장 블라인드제 도입 전/후**를 기준으로  
농산물 도매시장 거래 데이터를 분석하여  
**제도 도입이 가격 및 거래 행태에 미친 영향**을 정량적으로 살펴보는 것을 목표로 한다.

- 분석 대상 시장: **거래량 상위 5개 도매시장(서울가락/서울강서/부산엄궁/부산반여/대구북부)**
- 분석 품목: **사과**
- 분석 기준: **블라인드제 도입 전 / 도입 후 비교**

---

## 2. 분석 목적 💡

- 블라인드제 도입 전/후 **가격 수준 및 변동성 변화** 분석
- **거래량 추이** 비교를 통한 시장 반응 확인
- 시각화를 통한 정책 효과 직관적 해석

---

## 3. 데이터 수집 📡

### 3.1 데이터 출처

- 공공데이터포털
- 농산물 도매시장 거래 정보 Open API

### 3.2 수집 방식

- 시장 코드 및 품목 코드 기반 필터링
- 연 단위 자동 수집 스크립트 사용

### 3.3 수집 환경

- Language: Python
- 실행 환경: CMD

---

### 3.4 컴파일 및 실행

```bash
python get_data.py
```

## 4. 데이터 수집 코드 개요 🧑‍💻

- `requests` 라이브러리를 이용한 API 호출
- 날짜를 하루 단위로 증가시키며 반복 조회
- HTTP 오류, API 오류, 데이터 없음(NO_DATA) 예외 처리
- 수집 결과를 **엑셀 파일(.xlsx)** 형태로 저장

```text
0000시장_사과전체_연도.xlsx
```

## 5. Google Colab을 이용한 시각화 🧑‍💻

### 5.1 연구 질문
- **RQ1**. 블라인드 도입 후 평균 낙찰가격은 변화했는가?
- **RQ2**. 블라인드 도입 후 가격 변동성은 감소했는가?
- **RQ3**. 산지·출하자별 단가 편차는 감소했는가?

---

### 5.2 분석 방법
- 평균 비교: t-test
- 분산 비교: Levene test
- 시각화: Bar plot, KDE, Box plot
- 추가 분석:
  - 소농/대농 가격 비교
  - OLS 회귀분석 (등급·크기·포장·산지·출하자 통제)
  - Event Study (도입 전·후 가격 추이)

---

### 5.3 주요 결과
- 블라인드 도입 후 **평균 낙찰가격 상승**
- 가격 변동성 및 산지/출하자별 편차는 **감소하지 않음**
- OLS 결과:
  - 동일 조건 통제 시 블라인드 도입 효과 **+6,510원 (p < 0.001)**
- Event Study:
  - 도입 직후 가격 급변 없음 → 구조적 효과로 해석 가능

---

## 결론
블라인드 제도는 전체 평균 가격을 안정화하기보다는,  
**동일 조건 상품 간 공정한 가격 형성에 기여하는 구조적 제도**로 해석할 수 있다.
